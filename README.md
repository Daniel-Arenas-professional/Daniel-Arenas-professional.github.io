- Diseño responsive (compatible con dispositivos móviles)

- Navegación intuitiva con anclas a secciones
- Timeline interactivo de educación
- Barras de progreso para habilidades

## 📦 Cómo usar

1. **Local**: Abre el archivo `index.html` directamente en tu navegador
2. **GitHub Pages**: 
   - Sube este repositorio a GitHub
   - Ve a Settings → Pages
   - Selecciona "Deploy from a branch"
   - Elige la rama `main` (o `master`) y carpeta `/(root)`
   - Tu sitio estará disponible en `https://tu-usuario.github.io/nombre-repositorio`
